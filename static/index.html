API Documentation

            
            

                

                    
Endpoint

                    

                        GET https://your-api-domain.com/validate?email=test@example.com
                    

                


                

                    
Response Format

                    
{
    "email": "test@example.com",
    "is_valid": true,
    "provider": "example.com",
    "is_free": false,
    "is_disposable": false
}
